Data Cleaning using PySpark
What I Did

In this task, I worked on cleaning a dataset using PySpark. The data was not in a proper format, so I fixed different issues to make it usable for analysis.

I handled null values by replacing them with suitable values and also took care of blank fields. I converted text data into proper format by changing uppercase and lowercase where needed. I also corrected inconsistent values like country names and removed unwanted spaces.

What I Learned

From this, I understood how important data cleaning is before doing any analysis. I learned how to handle missing data, fix inconsistent values, and make the dataset clean and structured.

Outcome

After cleaning, the dataset became more accurate and consistent, and it is now ready for further processing and analysis.


SQL Queries – Joins and Aggregations
What I Did

In this task, I worked on SQL queries using two tables: Employee and Sales. I created both tables, inserted sample data, and performed different types of queries to understand how data can be analyzed.

I used GROUP BY to calculate total salary, average salary, maximum and minimum salary for each department. I also used HAVING to filter grouped data like departments with more employees or higher salary totals.

I performed JOIN operations (inner join and left join) to combine Employee and Sales data. Using this, I calculated total sales per employee, number of sales, product-wise sales, and department-wise sales.

I also worked on advanced queries like:

Finding top 3 employees based on sales
Finding employees with no sales
Calculating sales per department
Finding highest and average sales
Using subqueries to get highest values
What I Learned

From this, I learned how to use SQL for real data analysis. I understood how GROUP BY and HAVING work, how to combine tables using joins, and how to write queries to get meaningful insights from data.

Outcome

After performing these queries, I was able to analyze employee and sales data effectively and extract useful insights like top performers, department performance, and sales trends.
